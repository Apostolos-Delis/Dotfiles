# Explain Code

Explain how the specified file or code section works.

## Instructions

1. Read the file or code section specified by the user
2. Provide a clear explanation of:
   - The overall purpose of the code
   - Key functions/classes and their responsibilities
   - Important data structures
   - Flow of execution
   - Any notable patterns or techniques used
3. Keep the explanation concise but thorough
4. Use code references (file:line) when discussing specific sections
5. If the code has dependencies or integrates with other parts of the system, briefly explain those relationships
