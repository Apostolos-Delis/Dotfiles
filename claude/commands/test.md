# Generate Tests

Generate tests for the specified file or function.

## Instructions

1. Read the file or function specified by the user
2. Identify the testing framework used in the project:
   - Python: pytest, unittest
   - JavaScript/TypeScript: jest, vitest, mocha
   - Ruby: rspec, minitest
3. Generate comprehensive tests covering:
   - Happy path scenarios
   - Edge cases (empty inputs, null values, boundaries)
   - Error conditions
   - Any existing test patterns in the codebase
4. Follow the project's existing test file structure and naming conventions
5. Include setup/teardown if needed
6. Add descriptive test names that explain what is being tested
