# Explain Code

Explain how the specified file or code section works.

## Instructions

1. Read the file or code section specified by the user.
2. Explain:
   - Overall purpose
   - Key functions/classes and responsibilities
   - Important data structures
   - Flow of execution
   - Notable patterns or techniques
3. Keep the explanation concise but complete.
4. Use code references (`file:line`) when discussing specific sections.
5. If the code integrates with other parts of the system, briefly explain those relationships.
